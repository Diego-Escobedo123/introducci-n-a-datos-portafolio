# Portafolio - Introducción a Datos

Repositorio académico para el curso **Introducción a Datos**.

Este portafolio reúne proyectos, laboratorios, actividades y ejercicios desarrollados durante el curso, incluyendo análisis de datos, visualización, manejo de archivos, arquitectura de datos y simulaciones de sistemas de recompensas.

## Contenido

### Laboratorios
- Manejo de archivos en Python
- Visualización y análisis de datos
- Actividades con Pandas y metadata

### Caso de Estudio
Proyecto basado en una simulación de arquitectura de datos y sistema de recompensas inspirado en Amazon Prime, incluyendo:
- Generación de datos simulados
- Limpieza y transformación de datos
- Sistema de puntos
- Segmentación de usuarios
- Detección de churn
- Visualizaciones y análisis

## Tecnologías utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- HTML
- CSS
- GitHub Pages
- Google Colab

## GitHub Pages
Este repositorio también incluye una página web tipo portafolio desarrollada con HTML y CSS para visualizar los proyectos y actividades realizadas durante el curso.
